# Лабораторная работа №2
## Тема: Разработка экспертной системы для анализа рынка недвижимости.

**Как запустить:**
1. Убедитесь, что у вас установлен Python 3.8 или выше.
2. Клонируйте репозиторий на свой компьютер.
3. Установите необходимые зависимости с помощью команды:
   ```
   poetry install
   ```
4. Запустите приложение из папки src/ с помощью команды:
   ```
   cd src; python main.py
   ```
5. Следуйте инструкциям на экране для взаимодействия с экспертной системой.
