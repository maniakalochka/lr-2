# Лабораторная работа №2
## Тема: Разработка экспертной системы для анализа рынка недвижимости.

**Как запустить:**
1. Убедитесь, что у вас установлен Python3:
    ```
    python3 --version
    ```
2. Склонируйте репозиторий с кодом лабораторной работы:
    ```text
    git clone https://github.com/maniakalochka/lr-2.git
   ```
3. Запустите терминал и выполните следующие команды:
   ```bash
   cd src; python main.py
   ```
4.Следуйте инструкциям на экране для взаимодействия с экспертной системой.
